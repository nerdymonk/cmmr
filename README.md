## 🎯 What is This Project?

CMR is a TypeScript-based web application template that combines **TanStack Start** (a full-stack React framework) with modern UI components, styling, and deployment capabilities. It's designed to be production-ready and developer-friendly, making it perfect for:

- Building interactive dashboards
- Creating data visualization applications
- Developing real-time web applications
- Prototyping complex user interfaces

This project uses **Cloudflare Workers** for server-side processing, allowing you to deploy globally with minimal latency.

---

## ✨ Key Features

- **Modern React 19**: Latest version of React with improved performance and features
- **Full-Stack Framework**: TanStack Start handles both frontend and backend routing
- **Beautiful UI Components**: Pre-built components from Radix UI for accessibility
- **Type-Safe**: Written entirely in TypeScript for better code reliability
- **Data Visualization**: Built-in support for charts and graphs
- **Form Handling**: Integrated form management with validation (React Hook Form + Zod)
- **Responsive Design**: Tailwind CSS for beautiful, responsive layouts
- **Global Deployment**: Ready to deploy on Cloudflare Workers
- **Developer Experience**: Hot module reloading (HMR), ESLint, Prettier for code quality

## 🛠️ Technology Stack

### Frontend Technologies

| Technology | Purpose |
|---|---|
| **React 19** | Modern UI library for building interactive components |
| **TypeScript** | Adds type safety to JavaScript for fewer bugs |
| **Tailwind CSS** | Utility-first CSS framework for rapid styling |
| **Radix UI** | Unstyled, accessible component library (buttons, dialogs, menus, etc.) |
| **React Router (TanStack)** | Client-side routing for navigation between pages |
| **React Hook Form** | Lightweight library for form state management |
| **Zod** | TypeScript-first schema validation for forms and data |
| **Recharts & Chart.js** | Libraries for creating interactive charts and graphs |
| **Date-fns** | Utility library for date manipulation |
| **Lucide React** | Beautiful, consistent icon library |
| **Sonner** | Toast notifications (popup messages for users) |

### Backend & Infrastructure

| Technology | Purpose |
|---|---|
| **TanStack Start** | Full-stack React framework with server-side rendering |
| **Cloudflare Workers** | Serverless compute platform for running backend code globally |
| **Vite** | Lightning-fast build tool and development server |
| **Wrangler** | Cloudflare's CLI tool for managing and deploying Workers |

### Development Tools

| Technology | Purpose |
|---|---|
| **TypeScript** | Static type checking for JavaScript |
| **ESLint** | Identifies and fixes code quality issues |
| **Prettier** | Automatically formats code for consistency |
| **Bun** | Fast JavaScript runtime and package manager alternative to Node.js |
